# Rock-Paper-Scissors Game

A fun, interactive Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.

## Features
- Play best-of-N rounds against the computer
- Dark theme UI with real-time score tracking
- Responsive design

## Live Demo
[Play the game here](https://aadarsha-prog.github.io/rock-paper-scissors/projects/index.html)

## How to Play
1. Enter the number of rounds you want to play
2. Click "Start Game"
3. Choose Rock, Paper, or Scissors
4. See the result and your updated score

## Technologies Used
- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript
